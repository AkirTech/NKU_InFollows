欢迎来到 `NKU_InFollows` 项目！
本项目立足于一个现实问题：如何高效地收集整理信息，从中挖掘最重要的信息，让你的每一分钟花在刀刃上。（迫真）

# 项目特色
- 使用C++与Qt，采取Qml搭建前端界面，保证高效性与简洁美观。
- 结合Python在爬虫等方面的优势，优化代码。
- AI 驱动：利用AI技术对用户画像，进行个性化推荐。
- 与飞书紧密配合：一键添加任务，设置提醒，利用飞书的任务系统，追踪你的任务。(正在施工)
